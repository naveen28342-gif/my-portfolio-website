# ✅ Personal Portfolio Website — Naveen Kumar D

A clean, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed to showcase my education, skills, certifications, projects, resume, and internship goals in a professional web presence.

---

## 📌 About the Project

This portfolio website presents my profile as a BCA student interested in software development, AI, prompt engineering, and cloud technologies. It was built from scratch using core web technologies, with a focus on clarity, responsiveness, and a simple user experience.

> ⚡ **AI-Assisted, Not Vibe-Coded** — The layout, content, and structure were intentionally planned and reviewed by the developer. AI was used as a coding assistant, not as an autopilot.

## 🚀 Getting Started

No heavy framework or build setup is required.

### Run Locally

1. Clone this repository

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Start the local server

```bash
npm start
```

4. Open the site in your browser

```text
http://localhost:3000
```

---

## 📁 File Breakdown

### `index.html`

Contains the complete website structure, including the hero section, about section, skills, projects, education, certifications, contact links, and resume download button.

### `styles.css`

Handles the visual design of the website, including typography, colors, layout, cards, responsive behavior, hover states, and animations.

### `script.js`

Adds scroll reveal animations using the Intersection Observer API for a smoother browsing experience.

### `server.mjs`

Provides a lightweight Node.js server to run the portfolio locally on `localhost:3000`.
